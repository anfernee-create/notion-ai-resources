# Summarization Prompts

Comprehensive collection of AI prompts for summarizing content in Notion.

## Quick Summaries

### One-Line Summary
```
Summarize this in one sentence that captures the key point.
```

### Executive Summary
```
Create an executive summary with:
- Main objective
- Key findings (3-5 points)
- Recommended actions
```

### TL;DR Format
```
Provide a TL;DR that covers:
```

## Tips for Effective Summarization

1. **Specify length**: "in 3 sentences" or "under 100 words"
2. **Define audience**: "for executives" or "for technical team"
3. **Set format**: "as bullet points" or "in paragraph form"
4. **Request structure**: "with headers" or "chronologically"
5. **Add context**: "focusing on financial impact" or "highlighting risks"
