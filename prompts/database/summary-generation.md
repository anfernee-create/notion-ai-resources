# Summary Generation Prompts

## Overview

These AI prompts help generate concise, informative summaries for Notion database entries. Use them to create descriptions, abstracts, and overviews that make information scannable and accessible.

## Length-Specific Summaries

### One-Line Summary

**Prompt**: "Create a one-sentence (15-20 words) summary that captures the essential point of: [TEXT]"

### Tweet-Length Summary

**Prompt**: "Summarize in 280 characters or less, maintaining key information and readability: [TEXT]"

### Paragraph Summary

**Prompt**: "Generate a 3-5 sentence paragraph summary covering main points, key takeaways, and relevant details: [TEXT]"

### Executive Summary

**Prompt**: "Create executive summary (100-150 words) with:
- Core message
- Key findings
- Critical recommendations
- Business impact

Content: [TEXT]"

## Purpose-Specific Summaries

### Description Field

**Prompt**: "Generate database description (50-75 words) that:
- Explains what this is
- Highlights key aspects
- Indicates relevance/value
- Remains objective

Full content: [TEXT]"

### Preview Text

**Prompt**: "Create compelling preview text (30-50 words) that:
- Hooks reader interest
- Previews main value
- Encourages full read

For: [TEXT]"

### Search Snippet

**Prompt**: "Generate search-optimized snippet (150-160 characters) with key terms and clear value proposition: [TEXT]"

## Multi-Format Summaries

### Comprehensive Summary Package

**Prompt**: "Generate summary package for [TEXT]:

1. **Headline** (10 words)
2. **Tweet** (280 chars)
3. **Short** (50 words)
4. **Medium** (150 words)
5. **Key Points** (5 bullets)
6. **Quote** (Best pull-quote)"

**Prompt**: "Create progressive summary layers:

For: [TEXT]"

## Specialized Techniques

### Extractive Summary

**Prompt**: "Extract 3-5 most important sentences directly from text that capture core message: [TEXT]

Maintain original wording."

### Abstractive Summary

**Prompt**: "Create new concise description that captures essence without copying sentences directly: [TEXT]

Use original phrasing and synthesis."

### Bulleted Summary

**Prompt**: "Convert to bullet point summary:
- Main idea (1 bullet)
- Supporting points (3-5 bullets)
- Conclusion (1 bullet)

Text: [TEXT]"

### Comparative Summary

**Prompt**: "Summarize highlighting differences from [REFERENCE]:
- What's new/different
- What's similar
- Key distinctions
- Implications of changes

New content: [TEXT]"

## Quality Guidelines

### Summary Enhancement

**Prompt**: "Improve this summary: [SUMMARY]

Original text: [TEXT]

Make it:
- More concise
- More accurate
- More engaging
- Better structured"

### Summary Validation

**Prompt**: "Check if this summary accurately represents source:

Summary: [SUMMARY]
Source: [TEXT]

Identify:
- Missing critical info
- Inaccuracies
- Misleading statements
- Suggested improvements"

## Best Practices

1. **Capture essence** - Main idea first
2. **Be accurate** - Don't add information
3. **Stay objective** - Avoid opinion
4. **Use clear language** - Simple > complex
5. **Include keywords** - Aid searchability
6. **Match tone** - Align with source
7. **Specify length** - Control output size
8. **Test readability** - Ensure clarity
