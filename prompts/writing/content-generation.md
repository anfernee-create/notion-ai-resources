# Content Generation Prompts

A comprehensive collection of AI prompts for generating various types of content in Notion AI.

---

## Blog Posts & Articles

### Blog Post Outline
```
Create a detailed blog post outline about [TOPIC]. Include:
- Catchy title
- Introduction hook
- 5-7 main sections with subpoints
- Conclusion with call-to-action
- Target audience: [AUDIENCE]
- Tone: [TONE]
```

### Full Blog Post
```
Write a comprehensive 1500-word blog post about [TOPIC]. 
Target audience: [AUDIENCE]
Tone: Professional yet conversational
Include:
- SEO-optimized title
- Engaging introduction
- Clear section headings
- Actionable tips
- Relevant examples
- Strong conclusion
```

### Listicle Article
```
Create a listicle: "[NUMBER] [TOPIC] that [BENEFIT]"
For each item include:
- Clear heading
- 2-3 sentence description
- Why it matters
- Quick tip or example
```

### How-To Guide
```
Write a step-by-step guide on how to [GOAL].
Audience level: [Beginner/Intermediate/Advanced]
Format:
- Brief introduction
- Prerequisites needed
- Step-by-step instructions
- Common mistakes to avoid
- Next steps
```

## Combination Prompts

Combine multiple prompts for complex content:

```
1. First, generate an outline:
   "Create an outline for [TOPIC]"

2. Then expand each section:
   "Expand section 2 from the outline into 300 words"

3. Add supporting elements:
   "Create 3 examples for section 4"

4. Polish and refine:
   "Improve the flow between sections 2 and 3"
```

---

**Remember**: The more specific your prompt, the better the output. Always review and edit AI-generated content to ensure accuracy, brand voice, and quality.
