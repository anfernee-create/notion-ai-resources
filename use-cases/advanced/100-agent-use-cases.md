## How to Implement These Use Cases

Each use case follows the same pattern:
1. **Input**: Define the source data (text, database, external source)
2. **Prompt**: Write a specific Notion AI prompt
3. **Output**: Define where the result should go
4. **Trigger**: Decide when the agent runs (on creation, on schedule, on demand)
