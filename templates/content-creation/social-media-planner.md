# Social Media Planner Template

A comprehensive Notion template for planning, scheduling, and analyzing social media content across multiple platforms with engagement tracking and audience analytics.

## Overview

This template enables:
- Multi-platform content planning
- Content calendar and scheduling
- Post drafting and approval workflow
- Hashtag and topic management
- Engagement tracking and analytics
- Audience insights and growth monitoring
- Competitor analysis
- Campaign management

## Database Properties

### Social Posts Database

| Property | Type | Description | AI Autofill |
|----------|------|-------------|-------------|
| Post | Title | Post content/title | – |
| Platform | Multi-select | Instagram, Twitter, LinkedIn, TikTok, Facebook, Other | ✓ |
| Status | Select | Draft, Scheduled, Published, Archive | – |
| Content Type | Select | Image, Video, Carousel, Text, Poll, Story | ✓ |
| Publish Date/Time | Date | Scheduled publication date | – |
| Audience | Select | General, VIP, Followers, New | ✓ |
| Engagement Goal | Select | Awareness, Engagement, Conversion | ✓ |
| Hashtags | Text | Relevant hashtags | ✓ |
| Keywords | Text | SEO keywords | – |
| Author | Person | Content creator | – |
| Approver | Person | Content approver | – |
| Likes | Number | Post likes count | – |
| Comments | Number | Comment count | – |
| Shares/Retweets | Number | Share count | – |
| Click Rate | Number | CTR percentage | – |
| Campaign | Relation | Associated campaign | – |

## Usage Guide

### Planning Phase
1. Define monthly content themes
2. Identify key campaigns
3. Plan content mix (80/20 rule)
4. Schedule posting times
5. Determine platform focus

### Creation Phase
1. Write compelling copy
2. Select or create visuals
3. Research relevant hashtags
4. Add call-to-action
5. Draft post for review

### Scheduling Phase
1. Set publication date/time
2. Select target audience
3. Assign to approver
4. Optimize for platform
5. Schedule post

### Analysis Phase
1. Monitor real-time engagement
2. Track metrics daily
3. Identify top performers
4. Analyze audience sentiment
5. Adjust future strategy

## Best Practices

1. **Platform-Specific Content**: Tailor format to each platform
2. **Consistent Branding**: Maintain visual and voice consistency
3. **Optimal Posting Times**: Schedule when audience is most active
4. **Hashtag Strategy**: Mix popular and niche hashtags
5. **Engagement**: Respond to comments and messages promptly
6. **Visual Quality**: Use high-quality images and videos
7. **Call-to-Action**: Always guide viewers on next steps
8. **Analytics Driven**: Use data to refine strategy
9. **Frequency**: Maintain consistent posting schedule
10. **Authenticity**: Share genuine, valuable content

**Problem**: Audience growth plateauing

**Problem**: Inconsistent quality
