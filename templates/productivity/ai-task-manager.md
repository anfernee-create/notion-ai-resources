# AI-Powered Task Manager Template

Comprehensive task management system with AI auto-categorization, smart prioritization, and intelligent insights.

## Features
- Auto-categorize tasks by type
- AI-assigned priority levels
- Automated task breakdowns
- Daily/weekly AI summaries
- Smart next-action suggestions

## Setup

1. **Duplicate this template**
2. **Configure AI properties:**

```
Category (AI Autofill): "Categorize as: Work, Personal, Learning, Health, or Finance"
Priority (AI Autofill): "Assign High/Medium/Low based on urgency and keywords"
Summary (AI Autofill): "Summarize in one clear sentence"
```

3. **Customize views and start using!**

## Quick Prompts

**Break down project:**
```
Break [PROJECT] into actionable tasks with time estimates
```

**Weekly review:**
```
Analyze completed tasks this week. Generate:
- Top 5 accomplishments
- Progress analysis
- Next week priorities
```

**Daily plan:**
```
Optimize today's tasks by priority and energy level
```
