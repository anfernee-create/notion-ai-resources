# Best Practices: Writing Effective AI Prompts

## Overview

Effective prompt writing is crucial to getting the best results from Notion AI. This guide covers best practices, techniques, and examples for writing clear, specific prompts that yield accurate and useful responses.

## Fundamental Principles

### 1. Be Specific and Clear
- Use precise language
- Define scope clearly
- Specify output format
- Example: "Summarize this article in 3 bullet points" is better than "Summarize this"

### 2. Provide Context
- Include relevant background information
- Specify the audience
- Clarify the purpose
- Define any domain-specific terms

### 3. Set Clear Expectations
- Specify desired length
- Define tone and style
- Indicate format preferences
- State any constraints

## Prompt Structure

### Effective Formula
1. **Role/Context**: Define what the AI should act as
2. **Task**: Specify what you want it to do
3. **Details**: Provide specific requirements
4. **Format**: Specify the output format
5. **Constraints**: Add any limitations or rules

### Example Structure
```
You are a marketing expert. Write a social media post about [product] 
targeted at [audience]. The post should be [length] and include 
[specific elements]. Use [tone] language.
```

## Common Prompt Types

### Summarization
- Specify word count or bullet points
- Define key points to include
- State audience level

### Analysis
- Specify analysis type (SWOT, pros/cons, etc.)
- Provide data or context
- Define output structure

### Generation
- Define topic and scope
- Specify quantity and format
- Provide examples or guidelines

### Transformation
- Describe current format
- Specify target format
- Define transformation rules

## Examples

### Good Prompts
"Convert this meeting transcript into action items with assignees and deadlines"
"Write 5 unique email subject lines for a product launch, each under 50 characters"
"Explain quantum computing to a 10-year-old"

### Poor Prompts
"Summarize this"
"Write about AI"
"Make it better"

## Advanced Techniques

### Chaining Prompts
1. First prompt generates output
2. Second prompt refines or transforms
3. Third prompt optimizes further

### Few-Shot Prompting
- Provide 1-3 examples
- Show desired output format
- Help AI understand patterns

### Role-Based Prompting
- "You are a [role]"
- Helps AI adopt appropriate tone
- Provides context framework

## Common Mistakes to Avoid

1. **Vague instructions**: "Write something interesting"
2. **Conflicting requests**: Multiple contradictory goals
3. **Missing context**: Not enough background information
4. **Unclear scope**: Too broad or too narrow
5. **No format specification**: Leaving output ambiguous

## Tips for Better Results

- Test variations and iterate
- Be specific about examples
- Specify tone, style, and audience
- Define success criteria
- Use structured templates
- Provide feedback and refine
- Save effective prompts for reuse

## Prompt Templates

### Content Creation
"Write a [type] about [topic] for [audience], [specific requirements]"

### Analysis
"Analyze [data/text] from the perspective of [viewpoint], considering [factors]"

### Translation
"Convert [input format] to [output format], maintaining [key aspects]"

## Conclusion

Effective prompt writing is a skill that improves with practice. Start with clear, specific prompts and refine based on results.
