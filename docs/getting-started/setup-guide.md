### Step 2: Access Notion AI Settings

#### For Individual Users:

1. Open Notion on desktop or web
2. Click **Settings & Members** in the sidebar
3. Navigate to **My account**
4. Find the **Notion AI** section
5. Click **Try Notion AI**

#### For Team Admins:

1. Go to **Settings & Members**
2. Click on **Billing**
3. Find **Notion AI** add-on section
4. Click **Add Notion AI**
5. Choose billing cycle (monthly or annual)
6. Add payment method

### Step 3: Start Your Free Trial

Notion AI offers a free trial with limited responses:

- **Trial Period**: Varies by account
- **Trial Responses**: ~20-40 AI responses
- **No Credit Card Required**: For initial trial
- **Full Feature Access**: Try all AI capabilities

**To activate trial:**
```
1. Click any AI button or press Spacebar
2. Select "Try Notion AI"
3. Start using AI features immediately
```

### Step 4: Understanding Access Methods

There are multiple ways to access Notion AI:

#### Method 1: Spacebar Command
- Place cursor in any text block
- Press `Spacebar`
- Type your AI request
- Press `Enter` to generate

#### Method 2: AI Button
- Hover over any text block
- Click the **AI** button that appears
- Select from AI action menu

#### Method 3: Slash Command
- Type `/ai` in any block
- Choose from AI options
- Enter your prompt

#### Method 4: Selection Menu
- Highlight existing text
- Click **Ask AI** button
- Choose transformation option

### Step 5: Configure AI Settings

#### Personal Preferences

Customize your AI experience:

1. **Language Settings**
   - Default language for responses
   - Translation preferences

2. **Saved Prompts**
   - Favorite frequently used prompts
   - Access via AI menu

3. **Keyboard Shortcuts**
   - `Spacebar` for AI prompt
   - `Cmd/Ctrl + J` for quick AI access

#### Team Settings (Admin Only)

1. **Usage Policies**
   - Set guidelines for AI use
   - Data handling policies

2. **Billing Management**
   - Add/remove seats
   - View usage statistics
   - Manage subscription

3. **Member Permissions**
   - Control who has AI access
   - Monitor team usage

## Device-Specific Setup

### Desktop App (Mac/Windows)

1. **Download Latest Version**
   - Visit notion.so/download
   - Install latest version
   - Restart Notion

2. **Enable AI Features**
   - Log in to your account
   - AI features auto-sync from web

3. **Keyboard Shortcuts**
   - Mac: `Cmd + J` for AI
   - Windows: `Ctrl + J` for AI

### Web Browser

**Supported Browsers:**
- Chrome (recommended)
- Firefox
- Safari
- Edge

**Setup Steps:**
1. Go to notion.so
2. Log in to your account
3. No additional setup needed
4. AI features available immediately

### Mobile App (iOS/Android)

1. **Update App**
   - Install latest version from App Store/Play Store
   - Version 2.0+ required for full AI features

2. **Access AI on Mobile**
   - Tap the AI button in toolbar
   - Use `/ai` command
   - Limited features vs desktop

**Mobile Limitations:**
- Some AI features unavailable
- Database autofill limited
- Best experience on desktop

### Enable AI Database Properties

1. **Create or Open Database**
   - Any database type (table, board, gallery)

2. **Add AI Property**
   ```
   Click "+" to add property
   â†’ Select "AI Autofill"

### Example AI Properties

**Meeting Notes Summary:**
```

Source: Meeting notes page content
```

```

Source: Task description property
```

```
