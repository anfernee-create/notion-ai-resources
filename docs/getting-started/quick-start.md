## ‚ö° 5-Minute Quick Start

### Step 1: Enable Notion AI (1 minute)

1. Open any Notion page
2. Press `Spacebar` anywhere in a text block
3. Click **"Try Notion AI"** when prompted
4. Start your free trial (no credit card needed)

‚úÖ **Done!** You now have access to Notion AI.

### Step 2: Your First AI Prompt (2 minutes)

Try these three simple prompts:

#### Option A: Generate Content
```
1. Type: "Write a blog post outline about productivity tips"
2. Press Spacebar
3. Watch AI generate the outline
4. Edit as needed
```

#### Option B: Summarize Text
```
1. Paste any long text into Notion
2. Highlight the text
3. Click "Ask AI"
4. Select "Summarize"
```

#### Option C: Improve Writing
```
1. Write a few sentences
2. Highlight your text
3. Click "Ask AI"
4. Select "Improve writing"
```

### Step 3: Save Your Favorite Prompt (2 minutes)

## üé® AI Best Practices

### Be Specific
‚ùå **Vague**: "Write something about marketing"
‚úÖ **Specific**: "Write a 300-word blog intro about email marketing for small businesses"

### Provide Context
‚ùå **No context**: "Summarize this"
‚úÖ **With context**: "Summarize these meeting notes into 3 key decisions and 5 action items"

### Iterate and Refine
1. Generate initial output
2. Review what works
3. Ask AI to improve specific parts
4. Repeat until satisfied

### Use Follow-up Prompts
