# AI Connectors

## Overview

Notion's AI Connectors enable seamless integration between Notion and external applications, powered by intelligent automation. These connectors allow you to extend Notion's capabilities by connecting to popular third-party services and automating data flows between systems.

## Key Features

### 1. Intelligent Connection Setup
AI guides you through setting up connections with popular applications:
- Email providers (Gmail, Outlook)
- Communication tools (Slack, Teams, Discord)
- Project management (Asana, Monday.com, Jira)
- CRM systems (Salesforce, HubSpot)
- Cloud storage (Google Drive, Dropbox, OneDrive)

### 2. Automated Data Sync
Keep data synchronized between Notion and connected applications:
- Real-time updates
- Bidirectional synchronization
- Scheduled syncs
- Event-triggered actions

### 3. Smart Workflow Automation
Create intelligent workflows that:
- Monitor external events
- Trigger Notion updates automatically
- Send notifications based on conditions
- Update connected systems from Notion changes

## Supported Integrations

### Communication
- Slack: Send messages, create channels, manage threads
- Microsoft Teams: Post updates, create channel messages
- Discord: Send notifications to servers and channels

### Email & Calendar
- Gmail: Create events, add reminders
- Outlook: Sync calendars and tasks
- Email-to-Notion: Convert emails to database entries

### Project Management
- Asana: Sync tasks and projects
- Monday.com: Keep boards in sync
- Jira: Track issues and updates
- Trello: Mirror card information

### Cloud Storage
- Google Drive: Embed and sync files
- Dropbox: Monitor folder changes
- OneDrive: Sync documents
